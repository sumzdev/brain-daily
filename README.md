# BrainDaily - 전두엽 강화 훈련 서비스

숏츠 중독으로 약해진 전두엽을 매일 강화하는 인지 훈련 서비스입니다.

## 설치 및 실행

```bash
cd /Users/sumz/Study/brain-daily
npm install
npm start
```

## 주요 기능

- Stroop Test (색상 인지)
- N-back Game (작업 기억)
- Decision Making (의사결정)
- 요약 훈련 (정보 처리)
- 감정 라벨링 (감정 조절)

## 기술 스택

- React 18 + TypeScript
- Tailwind CSS
- Lucide React
- localStorage
